
MATRIZ.PY
================

Um programa que coloca um numero em uma certa posisao da matriz


----

Executar
----------

Efetuar a Execuçao do código-fonte:


       $ python3 matriz.py


----

Uso 
---
:

colocar um numero em uma tabela


----

Exemplos
--------
        1 |3| 2
        --|4|--
        6 |5| 7
